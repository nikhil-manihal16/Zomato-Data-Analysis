# Zomato-Data-Analysis
Analyzed Zomato restaurant dataset using Python to uncover customer preferences, restaurant trends, and pricing patterns. The project focused on extracting meaningful insights through data cleaning, exploratory data analysis (EDA), and visualization techniques to support data-driven decision making in the food industry.

<hr style="height:2px; background-color:black;">

<h2>🛠️ Technologies Used</h2>
  <ul>
  <li>Python</li>
  <li><b>Libraries:</b> Pandas, numpy, Matplotlib, Seaborn</li>
  <li><b>IDE:</b>Windsurf</li>
</ul>

<hr style="height:2px; background-color:black;">

<h2>🚀 Features</h2>
<ul>
  <li>Performed exploratory data analysis (EDA) on restaurant dataset</li>
  <li>Compared online vs offline delivery trends</li>
  <li>Analyzed popular restaurant types and cuisines</li>
  <li>Studied pricing patterns and customer preferences</li>
  <li>Generated actionable business insights</li>
</ul>

<hr style="height:2px; background-color:black;">

<h2>🔄 Data Preprocessing</h2>
<ul>
  <li>Handled missing values and null data</li>
  <li>Removed duplicate records</li>
  <li>Cleaned and standardized columns</li>
  <li>Converted data types for better analysis</li>
  <li>Filtered irrelevant features</li>
</ul>

<hr style="height:2px; background-color:black;">

<h2>📈 Visualizations</h2>
<ul>
  <li>Bar charts (restaurant types, pricing trends)</li>
  <li>Count plots (order types, ratings distribution)</li>
    <li>Heatmaps (correlation analysis)</li>
    <li>Boxplots (price vs ratings)</li>
</ul>

<hr style="height:2px; background-color:black;">

<h2>🔍 Key Insights </h2>
<ul>
  <li>Online food delivery services are more frequently used compared to offline dining options.</li>
  <li>Dining-type restaurants are the most popular among customers.</li>
  <li>The average spending for two people falls within a specific mid-range price category.</li>
  <li>Most restaurant ratings are concentrated between 3.5 and 4.0, indicating generally positive customer satisfaction.</li>
  <li>Customer preferences vary significantly based on cuisine type and location.</li>
</ul>
